# Banking Management System

## Overview

A simple yet powerful console-based banking management system built with Python and MariaDB.  
It supports creating accounts, deposits, withdrawals, and viewing transaction history.

---

## Features

- Create bank accounts with initial deposits  
- Deposit money into accounts  
- Withdraw money (with balance checks)  
- View account balances  
- View detailed transaction history  
- Robust input validation and error handling  
- Built with clean, modular Python code and MySQL database integration  

---

## Technologies Used

- Python 3  
- MariaDB / MySQL  
- mysql-connector-python  
- Termux (for Android environment)
